## 🧭 What this app does

Nano-Claude-Code is a small desktop tool that helps you work with code files. It is built for quick use on Windows and keeps setup simple.

Use it to:

- open code projects
- review code changes
- run common coding tasks
- work with a local project from your computer
- keep the interface light and easy to use

It is meant for people who want a simple way to run a coding assistant without a heavy setup.

## 🖥️ System requirements

This app works best on:

- Windows 10 or Windows 11
- 64-bit PC

## 🗂️ Tips for Windows users

- Keep the app in a folder that is easy to find.
- Use a short folder path, such as `C:\Apps\nano-claude-code`.
- Avoid running it from a temporary downloads folder.
- If Windows blocks the file, right-click it and check the file properties.
- If the app does not open, try running it again as an administrator.

## ⚠️ If something goes wrong

If the app does not start:

1. Check that the file finished downloading.
2. Make sure you extracted the `.zip` file if you downloaded one.
3. Try the latest file from the releases page.
4. Restart your PC and try again.
5. Move the app to a simple folder path.

If the window opens but looks blank:

1. Close the app.
2. Open it again.
3. Make sure your screen scaling is set to a normal value.
4. Try a different display setting if needed.

If your project does not load:

1. Check that you chose the right folder.
2. Make sure the folder contains the files you need.
3. Try opening a smaller project first.
